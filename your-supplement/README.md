# 🧬 당신의영양제 (Your Supplement)

객관적 근거(식약처 기능성 원료·논문) 기반 **개인 맞춤 영양제 추천** 서비스.
주관적 후기가 아니라 추적 가능한 근거로 추천하고, 복용 관리·알람·최저가까지 연결.

## 핵심 차별점
- ⭐ **근거 등급제**: 모든 추천에 출처(식약처/NIH/논문) + 신뢰도 ⭐⭐⭐ 표기
- ⏱️ **복용 기간 가이드**: 🟢지속 / 🟡점검 / 🔴주기 — "평생 먹어야 하나?" 해소
- 🔗 **상호작용 엔진**: 성분 시너지·길항·**의약품 경고** 자동 반영 → 개인 복용 스케줄
- 💰 **함량당 단가 최저가**: 단순 가격이 아닌 유효성분 mg당 가성비 랭킹
- 📦 **내 영양제 관리 + ⏰ 알람**: 카카오 알림톡 + 앱 푸시 이중 채널
- 💬 추천 결과 카카오톡 전송

## 구조 (웹·앱 공유)
```
your-supplement/
├── data/        # 시드 데이터(운영자 검수): 성분·고민·상호작용·추천규칙
├── engine/      # 추천 엔진(웹·앱·서버 공유 핵심 로직) — recommend.js
├── server/      # 백엔드 API + 스케줄러(알람·최저가 수집·카카오 발송)
│   └── api/
├── apps/
│   ├── web/     # Next.js (SEO·공유·결제)
│   └── mobile/  # React Native + Expo (푸시·복용관리·홈위젯)
└── README.md
```
> 추천 로직(`engine/`)은 한 벌만 두고 웹·앱·서버가 공유 → 추천 결과가 어디서나 동일.

## 데이터 모델
`data/schema.md` 참고. 정적 시드(성분/고민/상호작용/규칙) + 동적 DB(user/recommendation/review/price_offer/my_supplement/intake_schedule/kakao_message).

## 실행 (현재 검증 가능한 것)
```bash
node engine/demo.js     # 가상 사용자로 추천 엔진 결과 확인
```
