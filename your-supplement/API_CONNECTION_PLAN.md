# 🔌 데이터 API 연결 계획 (배포 후 실행판)

> 기준: 2026-06-30. 앱이 Vercel에 배포되어 **외부망이 열렸으므로** 아래 API가 실제 동작 가능.
> 공통 적용법: Vercel → 프로젝트 → **Settings → Environment Variables** 에 Key/Value 추가 → **Redeploy**.
> ⚠️ `NEXT_PUBLIC_` 변수(카카오맵)는 빌드 시 주입 → 추가 후 **반드시 재배포**.

---

## ✅ Tier 0 — 지금 이미 동작 (키 0개, 배포만으로 완료)

배포된 URL에서 **바로 실데이터**가 나옵니다. 할 일은 "확인"뿐.

| API | 라우트 | 가져오는 데이터 | 확인 방법 |
|-----|--------|----------------|----------|
| **NIH DSLD** | `/api/products` | 해외 시판 영양제 라벨 20만건+ | 제품찾기에서 성분(영문) 선택 → 실제 제품·라벨 링크 |
| **openFDA CAERS** | `/api/safety` | 성분별 이상사례 보고·상위 증상 | 제품찾기 결과의 ⚠️ FDA 이상사례 카드 |
| **NCBI PubMed** | `/api/evidence-search` | 성분별 임상연구(RCT·메타분석) | 제품찾기 결과의 📚 관련 임상연구 |

> 한글 자유검색도 주요 성분은 영문 매핑됨(아르기닌→arginine). `NCBI_API_KEY` 넣으면 rate만 상향(선택).

## 🟢 Tier 1 — 무료 키 5분 (즉시 실데이터, 승인 불필요)

가입 즉시 키가 나오는 것들. 사용자 체감 가장 큼(가격·지도).

### 1. 네이버쇼핑 — 성분별 최저가 (`/api/offers`)
- **발급**: developers.naver.com → 애플리케이션 등록 → **검색 API** 추가 → Client ID/Secret
- **Vercel 환경변수**: `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`
- **효과**: 추천 성분의 실시간 최저가/쇼핑 결과 (없으면 예시 가격)

### 2. 카카오맵 — 약국·응급실 지도 (`/nearby`)
- **발급**: developers.kakao.com → 앱 → 앱 키 → **JavaScript 키**
- **필수**: 플랫폼 → Web → **사이트 도메인에 배포 URL 등록** (안 하면 SDK 차단)
- **Vercel 환경변수**: `NEXT_PUBLIC_KAKAO_MAP_KEY` (→ 재배포 필수)
- **효과**: 목록만 → 실제 지도 임베드

## 🟡 Tier 2 — data.go.kr 활용신청 (승인 즉시~1일, 키 1개로 4개 해제)

공공데이터포털 계정 1개의 **같은 인증키(`DATA_GO_KR_KEY`)** 로 아래 서비스들을 각각 "활용신청"만 추가하면 됨.

### 3. 약국·응급실 실시간 (`/api/nearby`) — **먼저**
- **신청**: data.go.kr → 국립중앙의료원 「전국 약국/응급의료기관 정보 조회 서비스」 활용신청
- **환경변수**: `DATA_GO_KR_KEY`
- **효과**: 샘플 목록 → 실시간 전국 약국·응급실·병상

### 4. 국내 건강기능식품 품목 (`/api/products` 국내 섹션)
- **신청**: data.go.kr 「식품의약품안전처_건강기능식품정보」(데이터셋 15056760)
- **환경변수**: `MFDS_PRODUCT_API_URL` = 명세의 상세조회 엔드포인트 전체 URL
  (예 `http://apis.data.go.kr/1471000/HtfsItemInfoService/getHtfsItem`) + 위 `DATA_GO_KR_KEY`
- **효과**: 국내 시판 건기식 품목(제품→신고번호·업체) 실데이터

### 5. 식약처 DUR 병용금기 확장 (`/api/dur`)
- **신청**: data.go.kr 「의약품안전사용서비스(DUR)」(15059486)
- **환경변수**: `MFDS_DUR_API_URL` + `DATA_GO_KR_KEY`
- **효과**: 현재 21종 하드코딩 상호작용 → 수천종 공식 병용금기 조회

### 6. 식약처 기능성 원료 인정현황 (배치 적재)
- **신청**: data.go.kr 「건강기능식품 기능성 원료 인정현황」(15058359)
- **환경변수**: `MFDS_MATERIAL_API_URL` + `DATA_GO_KR_KEY`
- **실행**: `node scripts/sync-mfds-materials.mjs` → `data/mfds_raw_materials.json` 적재
- **효과**: 성분 KB·국내 인정문구(`kr_source`) 자동 확장 → deep-link `verified` 승격

## 🗺️ 권장 실행 순서

1. **Tier 0 확인** — 배포 URL에서 제품찾기 동작 확인(이미 됨).
2. **네이버 키** → 최저가(체감 큼, 5분).
3. **data.go.kr 키 발급 + 약국/응급실 신청** → `/nearby` 실데이터.
4. **식약처 품목(15056760)** → 국내 제품.
5. **카카오맵 키** → 지도(도메인 등록 후 재배포).
6. **식약처 DUR/원료** → 안전·근거 자동 확장.
7. (선택) LNHPD·알림톡.

> 모든 라우트는 **키 없으면 graceful 폴백**(샘플/딥링크)이라, 위를 **하나씩 켜는 즉시** 해당 영역만 실데이터로 바뀝니다. 한 번에 다 안 해도 됨.
