## 2. 근거 인용의 실태 (가장 중요)

- `data/evidence.json`: 17개 성분 전부에 **효능·용량·상호작용 + 인용 URL 75개**. 모두 검색 결과에 실재하는 링크.
- **인용 출처 호스트(전부 미국 기관)**: `ods.od.nih.gov`(NIH ODS), `nccih.nih.gov`(NCCIH), `nei.nih.gov`(국립안연구소), `medlineplus.gov`, `ncbi.nlm.nih.gov`(PMC).
- 근거 강도(`evidence_strength`) 표기: 확립 13 · 중등도 10 · 제한적 10 · 불충분 1 → **과장 없이 솔직**.

## 3. 신뢰도 갭 (개선 과제)

| # | 갭 | 영향 | 비고 |
|---|----|------|------|
| **D1** | **국내 1차 출처 인용 — 17종 전부 반영(2단계 표기)** — 식약처 기능성 내용을 성분별 `kr_source`로 **17종 모두** 추가. ① **deep-link 검증 3종**(루테인·프로바이오틱스·밀크씨슬, `verified:true`+원문 URL) ② **고시 표준문구 14종**(`verified:false`+`basis`, 「건강기능식품의 기준 및 규격」 고시·개별인정 표준 인정문구). 데모는 두 단계를 구분 표기("식약처 인정+원문" vs "식약처 고시 기능성+검색") | 국내 신뢰 보강(전체) | deep-link URL은 외부망 차단으로 미검증 → 배포 시 식품안전나라 OpenAPI로 자동 확인 예정 |
| **D2** | **ingredients.json 근거가 포괄적** — 17종 전부 `"기능성 원료 고시"` 한 줄. 구체 인정문구·고시번호·인정일 없음 | 1차 검수 미완 | evidence.json과는 연결됨 |
| **D3** | **약물 상호작용 21종으로 확장(전부 인용)** — 기존 12종 + 신규 9종(메트포르민→B12, PPI→마그네슘(FDA)·B12, 루프이뇨제→마그네슘·칼슘, 스테로이드→칼슘·비타민D, 경구피임약→엽산·B군, 항우울제(MAOI)→인삼, 면역억제제→프로바이오틱스·인삼, 디곡신→인삼) + 갑상선약×마그네슘 추가. 약물군 18종(고아였던 항생제 선택지 복구). 전 항목 NIH ODS·NCCIH·FDA URL 보유 | 안전 경고 대폭 강화 | 식약처 DUR 수천종 자동 확장은 배포 시 API(`MFDS_DUR_API_URL`)로 |
| **D4** | **수치(RDA/UL) 일부 '미설정'** — 루테인·프로바이오틱스·코엔자임Q10은 **공식 RDA/UL이 존재하지 않음**(영양소가 아닌 기능성 원료). 데모는 '출처 미확인' 대신 **"공식 권장량·상한 미설정"**으로 정직 표기(없는 상한 지어내지 않음) | 정밀도 한계(불가피) | 공식값 부재를 명시 |
| **D5** | **검수 거버넌스 부재** — 운영자 검수 전제이나 검수자·검수일·버전 이력 체계 없음 | 데이터 변경 추적 어려움 | `_meta.version`만 존재 |

## 4. 신뢰도 향상 권고 (우선순위)

1. **D3 → 약물 상호작용 확장(P5 DUR)**: 식약처 DUR(병용금기·노인주의 등)로 확장. 우선 evidence.json에 이미 수집된 **인용 있는 약물-영양제 상호작용**을 엔진에 반영(즉시 가능).
2. **D1 → 국내 1차 출처 채우기**: 식약처 기능성 원료 인정문구·고시번호를 성분별 `evidence.citations`에 추가. (자료 제공 또는 키 연동 시)
3. **D2 → ingredients 정밀화**: 각 성분 `evidence.sources`에 구체 고시 항목 연결.
4. **D5 → 검수 거버넌스**: 각 데이터 항목에 `reviewed_by`/`reviewed_at`/`source_checked` 필드 + CHANGELOG.

## 5. 결론

- **강점**: 효능 근거 100%가 추적 가능한 1차/공인 출처(75 인용), 후기 분리, 근거강도 솔직 표기, **국내 식약처 기능성 내용 17종 전부 반영(2단계 검증 표기)** → 동종 서비스 대비 신뢰 설계가 앞섬.
- **남은 약점**: ① deep-link 검증 14종 미완(D1, 배포 시 API 자동 확인) ② 약물 상호작용은 21종으로 확장했으나 식약처 DUR 수천종 대비 여전히 핵심군 중심(배포 시 DUR API로 자동 확장).
