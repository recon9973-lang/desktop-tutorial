# 🛒 영양제 구매처 연결 전략

"사람들이 영양제를 어디서 많이 사는지" + "우리 서비스에서 그곳과 연결(+수익화)하는 방법" 조사.

---

## 1. 한국인이 영양제를 많이 사는 채널 (순위 대략)

| 순위 | 채널 | 특징 | 제휴 가능성 |
|------|------|------|------------|
| 1 | **쿠팡** | 로켓배송, 국내 최대 이커머스. 영양제 구매 압도적 1위 | ✅ 쿠팡 파트너스 |
| 2 | **네이버쇼핑** | 가격비교 중심, 최저가 탐색의 시작점 | △ 가격비교 API |
| 3 | **아이허브(iHerb)** | 해외 직구 영양제 1위, 가성비·고함량 | ✅ 자체 제휴/리워드 |
| 4 | **올리브영** | 오프라인+온라인, 비타민·이너뷰티 | △ 쇼핑 큐레이터 |
| 5 | **약국/홈쇼핑** | 중장년층, 신뢰도 기반 | ✕ 직접 제휴 어려움 |

> 핵심: **최저가는 네이버쇼핑에서 비교 → 실제 구매는 쿠팡/아이허브에서** 하는 패턴이 많음.
> 그래서 우리 서비스는 "네이버로 최저가 비교 + 쿠팡/아이허브로 구매 연결" 조합이 자연스러움.

---

## 2. 연결(수익화) 방법

### 🅰️ 지금 단계 — 검색 딥링크 (구현 완료 ✅)
- API 키·제휴 가입 없이 **바로 동작**. 각 성분명으로 쿠팡·네이버·아이허브 검색 페이지를 새 탭으로 염.
- 코드: `apps/web/app/result/page.jsx`의 `buyLinks()` 함수
- 수익: 없음. 사용자 편의 + MVP 검증용.

### 🅱️ 다음 단계 — 쿠팡 파트너스 (제휴 수익)
- **쿠팡 파트너스**(partners.coupang.com) 가입 → 상품 딥링크 생성 → 구매 발생 시 수수료.
- 수수료: 기본 **3%**, 카테고리별 상이. **영양제는 식품계열이라 낮은 편**(정확한 율은 가입 후 확인 필수, 정책 수시 변동).
- 적용: `buyLinks()`의 쿠팡 URL을 **내 파트너스 추적 링크**로 교체하면 끝.
- 주의: "이 포스팅은 쿠팡 파트너스 활동으로 일정 수수료를 제공받습니다" **고지 문구 의무**.

### 🅲️ 보조 — 아이허브 제휴 / 리워드
- 직구 선호 사용자용. 아이허브 자체 제휴/추천 코드로 수수료 또는 할인 제공.
- 고함량·가성비 제품이 많아 "함량당 단가" 비교에서 유리하게 노출될 때가 많음.

### 🅳️ 정밀 최저가 — 네이버쇼핑 API (이미 구현)
- `server/api/offers.js`가 네이버쇼핑 API로 **함량당 단가** 최저가를 계산.
- 흐름: 네이버 API로 "최저 ₩○○ (mg당 ₩○)" **표시** → 구매 버튼은 쿠팡 파트너스/아이허브 제휴 링크.

---

## 3. ⚠️ 신뢰도 원칙 (가장 중요)

이 서비스의 정체성은 **"객관 근거 기반 추천"**. 제휴 수익이 추천을 왜곡하면 신뢰가 무너져요.

- ✅ **추천 순위 = 근거 점수로만** 결정 (제휴 수수료 높은 제품을 위로 올리지 않음)
- ✅ 구매처는 추천과 **분리된 영역**으로 표시 ("어디서 살지"는 사용자 선택)
- ✅ 제휴 링크 사용 시 **고지 문구** 노출 (법적 의무 + 투명성)
- ❌ "수수료 높은 제품을 1순위로" 같은 건 절대 금지

> 고약사 채널이 신뢰받는 이유와 같음: **추천의 근거와 돈의 흐름을 분리**.
