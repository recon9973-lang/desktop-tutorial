# VENOM 3단 카테고리 택소노미 (메인 → 하위 → 그 하위) 설계

> 사용자 지시: 좌측 카테고리 **메인(치과마케팅) → 하위(임플란트) → 그 하위**까지 모두 **독립 H1 페이지**.
> "마케팅" 있으면 마케팅 내용, 없으면 제목 그대로의 주제(환자 교육 수준). 모든 H1은 제작표준 6요구사항(3,000자+·인포그래픽·실사진·데이터·통계·섹션별 출처) 준수.
> 사이트 전체 적용.

## 레벨 정의
- **L1 메인** = 진료과목 마케팅 허브 (예: 치과마케팅) — *마케팅 관점* 콘텐츠. **이미 존재(7개)**.
- **L2 하위** = 일반 주제 (예: 임플란트, 치아교정) — *주제 그대로* 환자 교육 콘텐츠. **대부분 신규 제작 대상**.
- **L3 그 하위** = 구체 시술/세부 (예: 네비게이션 임플란트) — *주제 그대로*. **대부분 기존 스포크 재배치**.

## 중복 방지 원칙
- L2(일반)와 L3(구체)가 **의미상 동일**하면 별도 페이지를 만들지 않고 **기존 스포크를 L2로 승격**한다(중복 콘텐츠 금지).
- L2(일반)와 L3(구체)가 **명확히 구분**되면 L2 일반 페이지를 **신규 제작**하고 기존 스포크를 그 L3 자식으로 둔다.
- 표기: **[신규]** = L2 일반 페이지 신규 제작 / **[승격]** = 기존 스포크를 L2로 / **[기존]** = 이미 있는 H1.

## 신규 제작(L2 일반 페이지) 대상 정리
치과 5 · 피부과 3 · 정형외과 3 · 안과 1(시력교정) = **약 12개 신규 L2 페이지**.
나머지 L2는 **기존 스포크 승격**(라우트/사이드바 재배치, 콘텐츠 재사용)으로 중복 없이 처리.

## 실행 순서 (loop)
1. **[완료]** 글쓰기·SEO 스킬에 제작표준 6요구사항 + 계층 H1 규칙 반영.
2. 신규 L2 페이지 제작 — 치과(5) → 피부과(3) → 정형외과(3) → 안과(1). 각 제작표준 6요구사항 충족.
3. 승격 L2 — 라우트·사이드바 3단 아코디언·부모링크 재배치.
4. 좌측 사이드바 3단(허브→L2→L3) 구조로 개편.
5. **상단 네비게이션 정리** — 좌측 확정 구조에 맞춰 병원마케팅 드롭다운을 L2까지 노출(메가메뉴), 중복 제거.
6. 전수조사 — 전 H1 페이지 6요구사항 매트릭스 점검·수정.
