# VENOM 사이트 정보구조(IA) & 페이지별 H태그 — 분석 & 정리 (v2 완전판)

> 사장님이 확정한 H1 포함 전체 헤딩 구조를 SEO 관점에서 분석·정리.
> 원칙: **URL(문서)당 H1 1개 → H2(섹션) → H3(항목) → H4(세부)**, 순서 건너뛰기 금지(크기는 CSS).

## 0) 총평 — SEO 관점

| 항목 | 평가 |
|---|---|
| 정보구조(IA) | ✅ **허브-스포크 구조 우수** — 병원마케팅/AI마케팅/온라인마케팅 3개 허브 + 세부 스포크 |
| 헤딩 계층 | ✅ H1→H2→H3→H4 논리적, **순서 건너뛰기 없음** |
| 키워드/롱테일 | ✅ 시술·질환·광고상품 키워드 촘촘 → 검색·AEO·GEO 전방위 |
| E-E-A-T·AEO 신호 | ✅ 전문성·FAQ·두괄식·스키마 언급이 구조에 내장됨 |
| 페이지당 H1 1개 | ✅ 각 페이지 H1 1개 원칙 부합 |

## 3) 발견 & 실행 전 결정사항

1. **진료과목 9개 확정** → 배포된 `/implant`·`/orthodontics`는 **치과로 흡수하고 URL 제거**(301 또는 치과로 리다이렉트).
2. **신규 4개 페이지 생성**: 네이버검색광고·구글검색광고·공식채널관리·언론홍보 (URL·라우팅·메타·sitemap·링크).
3. **명칭 변경**: "홈페이지 제작" → **"SEO 홈페이지"** (나비·타이틀·본문·URL은 `/website` 유지 권장).
4. **URL 네이밍**: 현행 짧은 슬러그(`/dental`) vs 사장님 표기(`/dental-marketing`). 통일 필요.
5. **푸터 헤딩**: GEO 등 말미 "서비스/AI마케팅/회사정보" H4 = **공통 푸터** → 콘텐츠 H태그 계층에서 제외.
6. **허브 페이지**: 병원마케팅/AI마케팅/온라인마케팅은 H1 + 소개문 + 하위 스포크로의 내부링크(실제 `<a href>`) 구성.

## 4) 권장 실행 순서
1. 결정 1~4 확정(흡수·신규·개명·네이밍)
2. IA 반영: 신규 4페이지 라우팅/메타/sitemap 추가, 임플란트·교정 흡수
3. 페이지별 H1→H2→H3(→H4) 마크업을 이 spec대로 구현(순서 건너뛰기 0, 크기는 CSS)
4. 허브→스포크 내부링크(실제 href)로 토픽 클러스터 완성
5. 우리 SEO 도구로 각 URL 재점검(H1 1개·계층·title·메타)
