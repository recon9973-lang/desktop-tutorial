# 베놈 ERP 디자인 시스템 (프리미엄 샘플 기반)

참고 원본: `venom-wordpress/preview/sample-premium.html` (베놈 뷰티클리닉 "프리미엄형" — 가장 트렌디한 샘플)
방향: 프리미엄 샘플의 **브랜드 아이덴티티**(딥 바이올렛 그라디언트·글래스·라운드·Pretendard)를 가져오되,
ERP는 표·폼이 많으므로 **밝은 관리자 베이스 + 바이올렛 강조**로 각색(전면 다크 대신). 다크모드는 옵션.

---

## 1. 팔레트

프리미엄 샘플에서 추출:
- 브랜드 그라디언트: `#7c3aed → #4f46e5` (버튼/헤더/포인트)
- 밝은 액센트: `#c084fc → #818cf8`
- 다크 배경(히어로/사이드바용): `#0a0a0f`, `#120a24`
- 텍스트(다크 위): `#fff`, 보조 `#e8e0f0` / `#aaa`

ERP 토큰(밝은 관리자 각색):

| 토큰 | 값 | 용도 |
|---|---|---|
| `--brand` | `#6d28d9` (violet-700) | 주요 액션·링크 |
| `--brand-2` | `#4f46e5` (indigo-600) | 그라디언트 끝색 |
| `--brand-grad` | `linear-gradient(135deg,#7c3aed,#4f46e5)` | 버튼/헤더/활성탭 |
| `--brand-soft` | `#f5f3ff` (violet-50) | 카드 호버·선택 배경 |
| `--ink` | `#0d0a1a` | 본문 텍스트 |
| `--muted` | `#6b7280` | 보조 텍스트 |
| `--surface` | `#ffffff` | 카드/표 배경 |
| `--bg` | `#faf9fe` | 페이지 배경(살짝 보라끼) |
| `--border` | `#ece9f5` | 구분선 |
| 사이드바 | `#120a24` 배경 + 흰 글자 | 프리미엄 다크 네비 |

> 기존 컴포넌트의 `#533afd`는 `--brand`/`--brand-grad`로 교체.

## 2. 타이포그래피
- 폰트: **Pretendard** (프리미엄 샘플과 동일). 폴백 `sans-serif`.
- 제목: 600~700, 본문 400~500. 숫자(대시보드 KPI)는 tabular-nums.

## 3. 형태 / 컴포넌트
- 라운드: 카드 `12px`, 버튼 `10px`, 큰 히어로/뱃지 `40px`(pill).
- 그림자: 카드 `0 1px 3px rgba(24,16,48,.06)`, 강조 `0 8px 24px rgba(124,58,237,.15)`.
- **글래스 액센트**(대시보드 상단 요약 배너 등 제한적으로): 다크 그라디언트 + `backdrop-filter: blur(12px)` + `rgba(255,255,255,.08)` 보더.
- 버튼: primary = `--brand-grad` 흰 글자 / secondary = 흰 배경 + `--border` / danger = rose.

## 4. 화면별 적용
- **사이드바(거래처단/베놈단 2기둥)**: 다크(`#120a24`) 배경, 활성 항목 `--brand-grad` 바.
- **대시보드 KPI 카드**: 밝은 카드 + 업종/상태 색점. 상단에 글래스 요약 배너 1개(트렌디 포인트).
- **표(거래처/업무/정산)**: 밝은 surface, 얇은 `--border`, 행 호버 `--brand-soft`. 데이터 가독성 최우선(다크 금지).
- **업종/카테고리 색상 태그**: 마스터의 `colorTag` 사용, `색+투명도22%` 배경.
- **캘린더 스케줄러 블록**: `--brand-grad`, 과부하 경고는 rose.

## 5. 다크모드(옵션, 2차)
프리미엄 샘플이 원래 다크이므로, 다크모드는 그 팔레트(`#0a0a0f` 배경, 글래스 카드)를 거의 그대로 재사용 가능.

## 6. 적용 방법 (ERP 세션에서)
1. `styles/tokens.css`(아래 동봉)를 전역 CSS에 import + Pretendard 웹폰트 로드.
2. Tailwind 쓰면 `tailwind.config`의 theme.extend.colors에 brand 토큰 매핑.
3. 컴포넌트의 하드코딩된 `#533afd` → `var(--brand)` 또는 Tailwind `brand` 클래스로 치환.
