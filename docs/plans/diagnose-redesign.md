## 1. 깊은 리서치

### 1-1. 현재 페이지의 문제 (실측)
| 항목 | 현황 | 문제 |
|---|---|---|
| 입력 | 도구 6종이 각각 URL/키워드/질문을 따로 요구 | 원장이 같은 정보를 6번 입력해야 함 |
| 흐름 | "1분 자가진단 → 실측 체크 → 리포트" 3-스텝 버튼 | 버튼일 뿐, 실제로는 사용자가 도구를 하나씩 수동 실행 |
| 리포트 | 히어로 우측 스코어카드가 **"예시" 목업** | 진단 페이지의 얼굴이 가짜 숫자 — 신뢰 손상 |
| 자가진단 | 13항목 체크리스트 | 자기 응답 기반이라 '실측'이 아님 — 보조 수단인데 1번에 배치 |
| 도구 | SEO/GEO/키워드/트렌드/AEO/노출 6종 (전부 실API 연결됨) | 전문가용 도구가 첫 화면 주인공 — 원장에겐 과부하 |

### 1-2. 사용 가능한 실측 자산 (이미 연결·검증된 API)
| API | 엔드포인트 | 진단 활용 |
|---|---|---|
| 페이지 수집 | `/api/seo-proxy?type=fetch` | HTML+robots 수집 → SEOEngine 채점 |
| SEO 엔진 | `assets/seo-engine.js` `SEOEngine.analyze()` | 렌더와 분리된 순수 함수 — 재사용 가능 |
| 속도 | `/api/seo-proxy?type=psi` (Google PSI) | 모바일 성능·SEO·접근성 점수 |
| AI 실측 | `/api/insights?type=aeo` (Perplexity sonar) | "지역+진료과 추천" 질문 → 우리 병원 언급 여부 |
| 검색량 | `/api/seo-proxy?type=keyword` (네이버 검색광고) | "{지역}{진료과}" 월간 검색량 = 기회 크기 |

### 1-3. 벤치마크 관찰 (경쟁 진단 도구 패턴)
- NXT류 SEO 진단: **URL 1개 입력 → 자동 다항목 채점 → 종합점수 + 항목별 카드** 구조가 표준.
- 리드젠 관점: 진단 결과의 끝은 항상 **단일 CTA(상담/리포트 수령)** — 도구 나열이 아님.
- 원장(비전문가) 대상 진단은 점수보다 **"그래서 뭘 해야 하나"(처방)** 가 전환을 만든다.

## 3. 성실한 기획 (채택안 C 상세)

### 3-1. 퍼널 구조 (위→아래)
1. **히어로**: "우리 병원, AI와 검색에 얼마나 보일까요?" + **단일 폼(URL·진료과·지역) + [무료 정밀 진단 시작]**. 우측 스코어카드는 진단 완료 시 실데이터로 갱신("예시" 배지 제거).
2. **진행 스테이지**: 4단계 체크리스트가 순서대로 점등 — ①홈페이지 구조 수집 ②SEO 채점 ③AI 답변 실측 ④검색 수요 조회. 속도(PSI)는 병렬 시작, 도착 시 병합(20~60초 소요 안내).
3. **종합 리포트**: 종합점수 도넛 + 영역 카드 4(SEO 구조/모바일 속도/AI 노출/검색 기회) + **맞춤 처방 최대 4** + [PDF 저장] [카카오 상담] CTA.
4. **보조**: 1분 자가진단(13항목) — "실측 전에 감 잡기" 포지션으로 유지.
5. **전문가 도구 6종**: 기존 tool-card 그리드 그대로 유지(개별 심화 분석용 자산).

### 3-2. 채점·처방 규칙
- SEO 구조 = SEOEngine pct(속도 제외 — Google 기준 동일). 속도 = PSI performance.
- AI 노출 = Perplexity 실측: 도메인 인용 100 / 병원명 언급 70 / 미언급 25("기회"로 표현).
- 종합 = SEO 40% + 속도 25% + AI 35% (속도 미도착 시 SEO 53%+AI 47%로 재정규화).
- 처방(조건부): SEO<70→기술 정비 / 속도<60→이미지·스크립트 경량화 / AI<70→GEO 콘텐츠 구축 / 검색량≥1,000·AI<70→"월 {n}회 검색되는데 AI 답변에 우리 병원이 없음 — 선점 기회".
- 실패 허용: 각 단계 실패 시 해당 카드만 '측정 불가' 처리하고 나머지로 리포트 계속(전체 중단 금지).

### 3-3. 의료광고법·신뢰 가드
- 점수는 "공개 데이터 기반 자동 진단, 성과 보장 아님" 고지 유지(PDF 푸터 포함).
- 가짜 수치 금지 원칙: 목업 스코어카드는 진단 전 "예시" 배지 필수, 진단 후에만 실값 표시.
