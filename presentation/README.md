# 검색 환경의 대전환 — SEO·AEO·GEO 마케팅 마스터 클래스

주식회사 베놈(병원마케팅 대행사) 발표 자료. 3가지 포맷으로 제공합니다.

| 파일 | 용도 | 특징 |
|---|---|---|
| `SEO-AEO-GEO-마스터클래스-웹.html` | **웹/발표용 (클로드 디자인)** | 브라우저에서 바로 실행. 키보드(←/→/Space)·클릭·스와이프·F 전체화면. 화면 크기에 자동 맞춤. Pretendard 폰트 임베드 — 단일 파일로 공유 가능 |
| `검색환경의-대전환-SEO-AEO-GEO-마케팅-마스터클래스.pptx` | **수정·전달용** | PowerPoint에서 텍스트·표·도형 네이티브 편집 가능 |
| `SEO-AEO-GEO-마케팅-마스터클래스.pdf` | **인쇄·배포용** | 레이아웃 고정 PDF |

## 웹 버전 조작법

- **다음**: → / Space / 화면 오른쪽 클릭 / 왼쪽 스와이프
- **이전**: ← / 화면 왼쪽(28%) 클릭 / 오른쪽 스와이프
- **전체화면**: F (또는 우하단 ⛶ 버튼)
- **처음/끝**: Home / End
- 우측 점 네비게이션, 하단 슬라이드 카운터, 상단 진행률 바 제공

## 디자인 시스템

- 캔버스 `#f6f5f4` (웜 페이퍼) · 단일 액센트 `#0075de` (블루) · `#0f2440` (네이비)
- 1px 헤어라인, 그림자 없음, 고밀도 데이터 레이아웃
- 폰트: Pretendard
- 슬라이드 구성: [결론] 한 문장 + 고밀도 데이터 표/인포그래픽

> 웹 버전은 `scratchpad/build_web.py`로 `deck.html`(인쇄 소스)에서 생성됩니다.
