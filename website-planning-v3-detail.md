```

  1. 정의형 첫 문장 (AI 인용 최적화)
  2. 3~5문장 핵심 설명
  3. 관련 링크 1~2개

  GEO(Generative Engine Optimization)란 ChatGPT·Perplexity 등
```

## 7. 워드프레스 구현 가이드

### 7.1 추천 스택

| 항목 | 권장 선택 | 이유 |
|------|----------|------|
| 페이지 빌더 | **Elementor Pro** | next-t 수준 레이아웃 구현 가능 |
| SEO 플러그인 | **Rank Math Pro** | Schema 자동화, AI 크롤러 허용 |
| 캐싱 | **WP Rocket** | Core Web Vitals LCP 최적화 |
| 이미지 최적화 | **ShortPixel** | WEBP 자동 변환 |
| 폼 | **WPForms** or **Gravity Forms** | 상담 신청 폼 |
| 보안 | **Wordfence** | HTTPS·보안 (E-E-A-T) |
| 테마 | **Hello (Elementor 기본)** | 최경량 |

### 7.2 페이지 유형별 워드프레스 구현 방법

#### 홈페이지 — Elementor Pro
- Hero: Section + Heading + Button + Counter 위젯
- 비교 블록: 2컬럼 레이아웃 + Icon List 위젯
- Step 카드: Flip Box or Custom HTML
- 상담 폼: WPForms Elementor 위젯

#### 위키 페이지 — Custom Post Type + Elementor
```
CPT: "wiki" (병원마케팅 위키)
  - 제목, 본문, 저자, 카테고리
  - 커스텀 필드: 읽기시간, 핵심요약, 관련위키

Elementor 템플릿:
  - 2컬럼 (사이드바 TOC + 본문)
  - TOC: Table of Contents Plus 플러그인
  - FAQ 아코디언: Elementor Accordion + Rank Math FAQ Schema
  - 하단 크로스링크: Related Posts 위젯 (커스텀 쿼리)
```

#### "빠르게 핵심만 살펴보기" 구현
```php
// functions.php에 추가하거나 Elementor Custom HTML 위젯 사용
// TOC 플러그인 (Table of Contents Plus) + CSS 커스터마이징으로 구현
// 또는 Rank Math Pro의 내장 TOC 블록 활용
```

#### 프로세스 Step 카드 구현 (Elementor)
```
Icon Box 위젯 5개 → Flex Container (가로 배치)
각 카드: 번호 Icon + 제목 + 설명
Hover Effect: Entrance Animation (Fade Up)
모바일: 세로 스택
```

### 7.3 Schema 자동화 (Rank Math Pro)

```
설정 경로: Rank Math → Schema → Add New Schema

위키 페이지:
  - Article Schema: 자동 (Rank Math 기본)
  - FAQPage Schema: Rank Math FAQ 블록 사용 → 자동 생성
  - BreadcrumbList: Rank Math 브레드크럼 활성화

로컬 비즈니스:
  - LocalBusiness + MedicalBusiness:
    Rank Math → Local SEO 모듈 활성화
    사업자 정보 입력 → 자동 Schema 생성
```

### 7.4 llms.txt 워드프레스 업로드

```
방법 1: 루트 디렉토리에 직접 업로드 (FTP/FileManager)
  → /public_html/llms.txt

방법 2: Rank Math Pro 커스텀 페이지
  → Rank Math → 일반 설정 → 크롤 설정 → 커스텀 헤더 추가
```

### 7.5 내부링크 자동화

```
플러그인: Link Whisper
기능:
  - 신규 글 작성 시 관련 위키·서비스 페이지 내부링크 자동 제안
  - 고아 페이지(0개 내부링크) 자동 감지
  - 내부링크 보고서 (페이지별 링크 수 현황)

목표: 위키 페이지당 최소 5개 내부링크
```
