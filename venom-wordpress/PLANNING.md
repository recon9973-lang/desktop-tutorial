## 상세 페이지 레이아웃 원칙
- **왼쪽**: 전체 사이트맵 (고정 사이드바, 현재 위치 하이라이트)
- **오른쪽**: 해당 페이지 목차 (TOC, 스크롤 연동 하이라이트)
- **본문**: 제목(H1~H4) + 콘텐츠 + 관련 링크

## UX 흐름
1. 홈 → 서비스 개요 → 상세 서비스 → 문의하기
2. 블로그 → 콘텐츠 소비 → 신뢰 형성 → 문의하기
3. 검색 유입 → 상세 페이지 → TOC로 스캔 → 문의하기
