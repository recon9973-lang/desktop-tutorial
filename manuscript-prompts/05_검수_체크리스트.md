# 5단계 · 발행 전 검수 체크리스트 ✅

> 목적: 원고를 내보내기 전에 **톤 · 정확성 · 타겟 적합성 · 시각자료**를 마지막으로 점검해요.

---

## ✅ 자동 검수 프롬프트 (AI에게 셀프 점검 시키기)

```
아래 완성 원고를 발행 전 기준으로 냉정하게 점검해줘.
각 항목을 통과(O)/보완필요(△)/실패(X)로 표시하고, △·X는 어떻게 고칠지 한 줄씩 제안해줘.

[원고]
[ 완성 원고 + 이미지 캡션 붙여넣기 ]

[점검 항목]
1. 말투 — 다정한 구어체와 신뢰의 문어체가 둘 다 살아있나? 차갑거나 훈계조는 없나?
2. 타겟 적합 — 정한 타겟([ 직장인/환자 등 ])이 "내 얘기다" 느낄 도입부인가?
3. 안심 — 겁주거나 공포를 자극하는 표현은 없나?
4. 정확성 — 단정적 효능 보장(완치 100% 등)이나 과장은 없나?
5. 수치 — 모든 통계에 출처가 있나? 지어낸 숫자는 없나?
6. 시각화 — [📊 그래프] 자리에 이미지·캡션·대체텍스트가 모두 있나?
7. 구조 — 소제목이 질문형인가? 각 소제목 아래 즉답이 먼저 나오나?
8. 가독성 — 한 문단 3~4줄? 어려운 용어에 쉬운 풀이가 붙었나?
9. 마무리 — 따뜻한 응원 + 오늘 할 수 있는 행동 1가지 + 부드러운 CTA가 있나?
10. FAQ — 연관 질문 5개가 자연어 질문형으로 들어갔나?

마지막에 '발행 가능 점수(100점 만점)'와 가장 먼저 고쳐야 할 1가지를 알려줘.
```

## 🖐 사람이 직접 보는 최종 눈검사 (빠른 체크리스트)

- [ ] 소리 내어 읽었을 때 **친구가 설명해주는 느낌**인가?
- [ ] 타겟이 누구인지 글만 봐도 분명한가?
- [ ] 숫자·그래프가 **진짜 출처** 기반인가?
- [ ] 이미지가 모바일에서도 글자가 읽히는가?
- [ ] 의료광고 심의/표현 규정에 어긋나는 단정·보장 표현이 없는가?
- [ ] 마지막 문장이 따뜻하게 끝나며 다음 행동을 부드럽게 안내하는가?
