# 4단계 · 수치 → 그래프 & 인포그래픽 이미지 📊

> 목적: 원고 속 `[📊 그래프: ...]` 표시를 **실제 그래프·인포그래픽 이미지**로 바꾸는 단계예요.
> 글로만 있던 숫자가 한눈에 들어오는 그림이 되면, 신뢰도와 가독성이 확 올라갑니다.

---

## 1️⃣ 먼저: 어떤 수치를 어떤 그림으로? (데이터 정리 프롬프트)

```
아래 원고에서 시각화하면 좋을 수치/통계를 모두 찾아줘.
그리고 각각에 가장 잘 맞는 '그래프 유형'을 추천해줘.

[원고]
[ 3단계에서 쓴 원고 붙여넣기 ]

[해줄 일] — 표로 정리
| 위치(소제목) | 보여줄 수치 | 추천 그래프 유형 | 한눈 메시지 |
- 그래프 유형은 다음에서 골라줘:
  · 막대그래프(비교) · 꺾은선(추이·변화) · 원형/도넛(비율)
  · 픽토그램(사람 아이콘으로 비율) · 비교 카드(전/후) · 타임라인
- '한눈 메시지'는 그 그래프를 보고 독자가 5초 안에 느껴야 할 한 문장.
- 출처가 필요한 수치는 [출처 필요]라고 표시해줘. (가짜 통계 절대 만들지 말 것)
```

> ⚠️ **중요**: 의학·건강 수치는 반드시 신뢰할 수 있는 실제 출처(보건 통계, 학회 자료 등)를 확인하세요. 없는 숫자를 지어내지 않습니다.

---

## 2️⃣ 이미지 생성 프롬프트 (인포그래픽/그래프 이미지)

> 이미지 생성 AI(예: `generate_image` 류)에 그대로 넣는 프롬프트예요. `[ ]`만 채우면 됩니다.

```
Clean medical infographic, single [bar chart / donut chart / line graph / pictogram].
Data to show: [ 예: 직장인 10명 중 7명이 거북목 증상 경험 → 70% ].
Headline text (Korean, large, friendly): "[ 한눈 메시지 한 문장 ]".
Style: 다정하고 깔끔한 헬스케어 톤, 부드러운 파스텔 + 포인트 컬러 [ 브랜드 색 ],
둥근 모서리, 넉넉한 여백, 군더더기 없는 플랫 디자인, 한국어 라벨,
신뢰감 있지만 따뜻한 느낌, 사람 아이콘 사용 가능.
No clutter, no fake logos, legible Korean typography, high contrast for readability.
Aspect ratio: [ 1:1 (카드형) / 4:5 (모바일 세로) / 16:9 (블로그 가로) ].
```

### 유형별 한 줄 변형 예시

| 보여줄 것 | 그래프 | 프롬프트 핵심 문구 |
| :--- | :--- | :--- |
| 비율(몇 %) | 도넛/픽토그램 | `donut chart, 70% highlighted, "10명 중 7명"` |
| 비교(A vs B) | 막대 | `two-bar comparison, before vs after` |
| 시간 변화 | 꺾은선 | `line graph trending down over 3 months` |
| 단계/과정 | 타임라인 | `horizontal timeline, 4 steps with icons` |

## 3️⃣ 글 + 이미지 합치기 (배치 프롬프트)

```
원고의 [📊 그래프] 표시 자리에, 방금 만든 이미지를 어디에 넣을지
'이미지 캡션(설명 한 줄)'과 함께 배치안을 만들어줘.
- 캡션도 다정한 구어체 + 문어체 톤으로.
- 이미지 바로 앞 문장이 그래프를 자연스럽게 소개하도록 연결 문장도 한 줄 제안해줘.
- 대체텍스트(alt text)도 접근성을 위해 한 줄씩 달아줘.
```
