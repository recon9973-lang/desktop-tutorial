# ★ 올인원 통합 프롬프트 (한 번에 원고 한 편) 🚀

> 0~5단계를 따로 돌릴 시간이 없을 때, **이 프롬프트 하나로 기획부터 시각화 설계까지** 한 번에 끝냅니다.
> `[ ]` 빈칸만 채워서 AI에 붙여넣으세요.

---

```
너는 병원·헬스케어 콘텐츠를 기획하고 집필하는 베테랑 에디터야.
아래 정보로 '브레인스토밍 → 주제·타겟 확정 → 원고 → 수치 시각화 설계'까지 한 흐름으로 진행해줘.

━━━━━━━━━━━━━━━━━━
[입력 정보]
- 분야/진료과목: [ 예: 정형외과 ]
- 알리고 싶은 핵심: [ 예: 직장인 거북목 ]
- 타겟(누구를 위한 글): [ 예: 하루 8시간 앉아 일하는 30대 직장인 / 또는 ○○ 환자 / 보호자 ]
- 채널: [ 블로그 / 뉴스레터 / 홈페이지 칼럼 ]
- 분량: [ 예: 1,500자 ]
━━━━━━━━━━━━━━━━━━

[말투 규칙] (반드시 준수)
1. '다정한 구어체 + 신뢰의 문어체'를 섞는다.
   · 공감·말걸기 = 구어체("~하시죠?", "~거든요", "~해요")
   · 정보·통계·치료법 = 문어체("~입니다", "~합니다")
2. 독자를 아끼는 사람처럼 대하고, 겁주지 않고 안심시킨다.
3. 어려운 의학 용어는 괄호로 쉽게 풀어준다. (예: 경추(목뼈))
4. 모호한 수식어 대신 구체적 수치를 쓴다. (평균 3개월, 약 30%)
5. 한 문단 3~4줄. 과장·효능 보장(완치 100% 등)·공포마케팅 금지.

[진행 순서] — 각 단계 결과를 보여주고 진행해줘
① 브레인스토밍: 타겟이 검색할 자연어 질문 10개 → 그중 원고 1편으로 좋은 주제 3개 추천(가제 포함).
   내가 하나 고를게.
② 타겟 정밀화: 고른 주제의 페르소나(불안 3가지·궁금증 3가지·유도할 행동 1가지) 정리
   + 최종 제목 3안 + 질문형 소제목 목차.
③ 원고 집필: 위 조건과 말투 규칙대로 본문 작성.
   - 도입부는 타겟의 일상 장면으로 공감하며 구어체로 시작.
   - 소제목 아래 즉답형 답변 먼저, 그다음 쉬운 설명.
   - 수치가 들어갈 자리엔 [📊 그래프: (어떤 데이터인지)] 표시만 남길 것.
   - 마무리는 따뜻한 응원 + 오늘 할 행동 1가지 + 부드러운 CTA.
   - 끝에 FAQ 5개(질문형).
④ 시각화 설계: 본문의 [📊 그래프] 자리마다
   '추천 그래프 유형 + 한눈 메시지 + 이미지 생성 프롬프트(영문)'를 표로 정리.
   (수치는 실제 출처 필요한 것에 [출처 필요] 표시, 가짜 통계 금지)

먼저 ①번부터 시작해줘. 내가 "다음" 하면 ②로 넘어가자.
한꺼번에 쏟아내지 말고 옆에서 같이 가듯 단계별로 진행해줘.
```

Data: [ 한눈 메시지 + 수치 ].

Style: 따뜻하고 깔끔한 헬스케어 톤, 파스텔 + 포인트 컬러 [ 브랜드색 ],
둥근 모서리, 넉넉한 여백, 플랫 디자인, 가독성 높은 한국어 라벨.
