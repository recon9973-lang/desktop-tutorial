# 2단계 · 말투 & 문체 가이드 (다정한 구어체 + 신뢰의 문어체) 💬

> 목적: 글의 **톤을 규칙으로 고정**하는 단계예요.
> "옆에서 다정하게 설명해주되, 내용은 전문가답게 믿음직한" 톤을 만들어요.

---

## 🎨 우리가 잡는 톤 한 줄 정의

> **다정한 구어체로 말 걸고, 핵심 정보는 문어체로 또박또박.**
> 친구처럼 다가가되, 의학 정보는 추측 없이 단정적으로 — 이 둘을 한 글 안에서 섞습니다.

| 구분 | 다정한 구어체 (말 거는 부분) | 신뢰의 문어체 (정보 전달 부분) |
| :--- | :--- | :--- |
| 쓰는 곳 | 도입부, 공감, 질문 던지기, 마무리 | 증상 설명, 통계, 치료법, 결론 |
| 어미 | "~죠", "~거든요", "~해요", "~인데요" | "~입니다", "~합니다", "~필요합니다" |
| 느낌 | 따뜻하고 부드럽게 | 정확하고 단정하게 |
| 예시 | "요즘 목이 자주 뻐근하시죠?" | "거북목은 경추 정렬이 무너진 상태를 말합니다." |

---

## ✅ 복사해서 쓰는 프롬프트

```
지금부터 쓸 모든 원고는 아래 '말투 규칙'을 반드시 지켜줘.

[말투 규칙]
1. 톤은 '다정한 구어체 + 신뢰감 있는 문어체'를 섞는다.
   - 독자에게 말을 걸거나 공감할 땐 구어체: "~하시죠?", "~거든요", "~해요"
   - 증상·통계·치료법 등 핵심 정보를 전할 땐 문어체: "~입니다", "~합니다"
2. 독자를 '환자/고객'이 아니라 '내가 아끼는 사람'처럼 대한다. 겁주지 않고 안심시킨다.
3. 어려운 의학 용어는 쓰되, 반드시 쉬운 말로 한 번 풀어준다.
   예: "경추(목뼈)가요..." 처럼 괄호로 친절하게.
4. 추측성 표현(아마, ~일 수도) 대신, 정보 문장은 단정형으로 신뢰감 있게 끝낸다.
5. 모호한 수식어('빨리', '많이') 대신 구체적 수치를 쓴다. ('평균 3개월', '약 30%')
6. 한 문단은 3~4줄 이내로 짧게. 호흡이 편하게.
7. 과장·공포 마케팅, 단정적 효능 보장(100% 완치 등)은 절대 금지.

[지켜야 할 금기]
- 차갑고 사무적인 톤
- 영혼 없는 '~됩니다' 반복
- 독자를 가르치려 드는 훈계조

이 규칙을 이해했으면 "톤 세팅 완료"라고 답하고,
예시로 [ 주제 ]에 대한 도입부 3문장만 이 톤으로 써서 보여줘.
```

## 🧪 톤이 잘 나왔는지 확인하는 미니 테스트

```
방금 쓴 도입부를 보고 스스로 점검해줘.
- 구어체 문장과 문어체 문장이 둘 다 들어갔나? (각각 표시)
- 겁주지 않고 안심시키는 느낌인가?
- 추측성 어미가 남아있진 않나?
부족하면 한 번 더 다듬어줘.
```

> 이 가이드는 3·5단계에서 계속 재사용됩니다. 원고를 쓸 때마다 위 [말투 규칙] 블록을 함께 붙여넣으세요.
