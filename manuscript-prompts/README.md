# 원고작성 프로젝트 — 프롬프트 + 원고 스튜디오 ✍️

병원·헬스케어 콘텐츠를 **기획(브레인스토밍) → 집필 → 이미지/카드뉴스 → 키워드 → 검수 → 발행**까지
한 흐름으로 끌고 가는 도구 모음입니다. 두 가지로 쓸 수 있어요.

- **① 프롬프트 세트** (`00~05`, `master`) — 복사해서 아무 AI에 붙여넣는 단계별 프롬프트
- **② 원고 스튜디오** (`studio.html`) — 더블클릭하면 브라우저에서 바로 도는 단일 파일 웹앱

> 빠르게 보려면 **`studio.html`을 더블클릭**하세요. 서버 없이도 대부분 기능이 작동합니다.

## 📌 이 세트의 콘셉트

- **말투**: 다정한 구어체 + 신뢰감 있는 문어체를 섞은 톤 (옆에서 설명해주는 전문가 느낌)
- **주제 / 주제가 누구**: 무엇을, *누구를 위해* 쓰는지 먼저 못 박고 시작
- **타겟**: 직장인 · 환자(질환별) · 보호자 등 구체적 페르소나 설정
- **수치 시각화**: 통계·수치는 글로만 두지 않고 **그래프 / 인포그래픽 이미지**로 전환
- **흐름**: 브레인스토밍 기획 → 주제·타겟 확정 → 문체 → 원고 → 시각화 → 검수

## ⚙️ 서버 연동 설정 (선택)

서버 기능을 쓰려면 Vercel 배포 + 환경변수가 필요합니다. 스튜디오 STEP3 **⚙️ 연결 설정**에 API 주소·`ADMIN_SECRET`을 넣어두면 모든 서버 기능이 공유합니다.

### 백엔드 엔드포인트
| 엔드포인트 | 기능 | 필요 환경변수 |
| :--- | :--- | :--- |
| `POST /api/generate-post` | 원고 생성(타겟·톤 반영) | `OPENAI_API_KEY` (+`ADMIN_SECRET`) |
| `POST /api/generate-image` | AI 이미지(base64) | `OPENAI_API_KEY` |
| `POST /api/keywords` | 연관어·**월 검색량** | (검색량) `NAVER_AD_API_KEY`·`NAVER_AD_SECRET`·`NAVER_AD_CUSTOMER_ID` |
| `POST /api/wp-draft` | WordPress 임시저장/예약/발행 | (요청 본문에 사이트·앱비밀번호) |

### WordPress 발행 설정 (1분)
1. 워드프레스 관리자 → **사용자 → 프로필 → 응용프로그램 비밀번호(Application Password)** 생성
2. 스튜디오 STEP5 → ⚙️ 워드프레스 연결 → 사이트 URL·사용자명·앱 비밀번호 입력
3. 발행 모드 선택(**임시저장 / 예약 / 즉시**) → 버튼 한 번

> ⚖️ **약관 안내:** 자동 발행/임시저장은 **공식 API가 있는 WordPress(내 소유 사이트)에서만** 안전합니다.
> 네이버 블로그·티스토리는 자동 등록이 약관 위반(계정 위험)이라, 스튜디오는 **붙여넣기용 내보내기**(복사→붙여넣기→임시저장)만 제공합니다.
