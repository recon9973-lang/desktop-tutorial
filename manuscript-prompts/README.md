# 원고작성 프로젝트 프롬프트 모음 ✍️

병원·헬스케어 콘텐츠 원고를 **기획(브레인스토밍)부터 완성**까지 한 흐름으로 끌고 가는 프롬프트 세트입니다.
각 파일은 그대로 복사해서 AI에게 붙여넣으면 바로 쓸 수 있도록 만들어 두었어요.

## 📌 이 세트의 콘셉트

- **말투**: 다정한 구어체 + 신뢰감 있는 문어체를 섞은 톤 (옆에서 설명해주는 전문가 느낌)
- **주제 / 주제가 누구**: 무엇을, *누구를 위해* 쓰는지 먼저 못 박고 시작
- **타겟**: 직장인 · 환자(질환별) · 보호자 등 구체적 페르소나 설정
- **수치 시각화**: 통계·수치는 글로만 두지 않고 **그래프 / 인포그래픽 이미지**로 전환
- **흐름**: 브레인스토밍 기획 → 주제·타겟 확정 → 문체 → 원고 → 시각화 → 검수

| 1 | [`01_주제_타겟_선정.md`](./01_주제_타겟_선정.md) | "이 글은 누구를 위한 글인가"를 확정해요 (직장인·환자 등) |

## 🚀 빠른 사용법

1. `00 → 05` 순서대로 프롬프트를 복사해 AI에 붙여넣고, `[ ]` 표시된 빈칸만 내 상황에 맞게 채워요.
2. 한 단계 결과가 마음에 들면 다음 단계 프롬프트의 입력값으로 넘겨요.
3. 빠르게 한 편을 뽑고 싶다면 `master_올인원_프롬프트.md` 하나만 써도 됩니다.

> 💡 Tip. 같은 주제라도 **타겟(누구)**만 바꾸면 전혀 다른 글이 됩니다. 직장인용 → 환자용 → 보호자용으로 변주해 보세요.
