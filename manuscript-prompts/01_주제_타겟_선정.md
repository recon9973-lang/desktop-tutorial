# 1단계 · 주제 & 타겟(누구를 위한 글인가) 확정 🎯

> 목적: "무엇을 쓸지(주제)"와 **"누구를 위해 쓸지(타겟)"**를 못 박는 단계예요.
> 같은 주제라도 *직장인용*과 *환자용*은 완전히 다른 글이 됩니다. 여기서 방향이 정해져요.

---

## ✅ 복사해서 쓰는 프롬프트

```
너는 독자 분석에 강한 헬스케어 콘텐츠 전략가야.
아래 주제로 원고를 쓸 건데, '누구를 위한 글인지'부터 같이 정밀하게 잡아줘.

[주제]
- 가제: [ 0단계에서 고른 제목 ]
- 핵심 메시지: [ 이 글로 결국 전하고 싶은 한 문장 ]

[타겟 후보] (해당되는 것 표시 / 추가)
- [ ] 직장인 (앉아서 일하는 / 야근 잦은 / 30~40대 등)
- [ ] 환자 본인 (예: 만성통증, 고혈압, 불면 등 질환 적기 → [   ])
- [ ] 보호자/가족
- [ ] 건강검진·예방 관심층
- [ ] 기타: [   ]

[해줄 일]
1. 위 후보 중 '이 주제에 가장 잘 맞는 1순위 타겟'을 추천하고 이유를 알려줘.
2. 그 1순위 타겟의 페르소나를 구체적으로 그려줘.
   - 한 줄 인물 소개 (이름·나이·상황 가상으로)
   - 이 사람이 지금 느끼는 불편/불안 3가지
   - 이 사람이 진짜 듣고 싶은 말 / 가장 궁금한 질문 3가지
   - 이 사람을 움직이게 할 '행동 한 가지' (예약, 자가진단, 습관 바꾸기 등)
3. 이 타겟에게 맞춘 '최종 제목 후보' 3개를 다정한 구어체로 제안해줘.
4. 글에서 다룰 핵심 메시지를 '도입-본문-마무리' 뼈대(목차)로 잡아줘.
   - 본문 소제목은 독자가 검색할 법한 질문형으로 (H3 느낌)
   - 수치/통계로 보여주면 좋을 지점에는 [📊 그래프] 라고 표시해줘.

결과는 표와 목록으로 깔끔하게 정리해줘.
```

---

## 🧩 결과 활용법

- 확정된 **타겟 페르소나 + 최종 제목 + 목차([📊 그래프] 표시 포함)**를 다음 단계로 넘겨요.
- `[📊 그래프]` 표시는 4단계([`04_수치_그래프_인포그래픽.md`](./04_수치_그래프_인포그래픽.md))에서 실제 이미지로 바뀝니다.

## 🔁 타겟만 바꿔 변주하기 (재활용 프롬프트)

```
방금 잡은 글을, 타겟만 [ 직장인 → 환자 본인 / 보호자 ]로 바꿔서
제목·도입부 후크·말투를 어떻게 다르게 가져가야 할지 비교표로 보여줘.
```
