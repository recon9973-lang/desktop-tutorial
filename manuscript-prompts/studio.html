✍️ 원고 스튜디오

분야·핵심·타겟만 넣으면 — 다정한 헬스케어 원고와 인포그래픽까지, 한 흐름으로
STEP 0 · 시작

무엇을, 어떻게 쓸까요?

기본 정보와 옵션을 정하면 이후 모든 단계가 여기에 맞춰집니다. 처음이라면 우측 상단 🎲 예시로 한 번에 채워보세요.
STEP 1 · 브레인스토밍

🪄 아이디어 스파크

핵심 키워드로 검색 질문을 3가지 의도로 나눠 펼치고, 원고 주제 후보를 카드로 제안합니다. 마음에 드는 카드를 고르세요.

🔎 키워드 리서치 (연관어·월 검색량)

핵심 키워드의 연관 검색어·자주 묻는 질문·실제 월 검색량을 가져옵니다(네이버/구글). 키워드를 누르면 보조 키워드로 담겨요. ⚙️ 엔진 연결 시 검색량까지 표시.

✂️ 본문에서 키워드 추출

원고·자료를 붙여넣으면 자주 나온 핵심어를 뽑아줍니다(브라우저에서 즉시, 검색 불필요).
STEP 2 · 타겟 설정

🎯 이 글은 누구를 위한 글인가요?

타겟을 바꾸면 같은 주제도 전혀 다른 글이 됩니다. 고르면 페르소나가 자동으로 채워지고, 직접 다듬을 수도 있어요.
✏️ 페르소나 직접 편집 (선택)
STEP 3 · 톤 & 프롬프트

🎚️ 톤 믹서 & 올인원 프롬프트

다정함과 전문성의 비율, 그리고 구성 옵션을 정하면 프롬프트가 실시간으로 조립됩니다. 어떤 AI에 붙여넣어도 작동해요.

    


    

      
🚀 베놈 엔진으로 실제 원고 생성 (베놈 사이트와 동일 파이프라인)

      
버튼 한 번으로 베놈 엔진(/api/generate-post)을 호출합니다 — 10단 구조·의료광고 자동검증·인포그래픽·FAQ·이미지까지. ⚙️ 연결 설정

      

        

          

          

        

        

          
          
          
        

        
설정은 이 브라우저에만 저장돼요(localStorage). 키는 서버로만 전송됩니다.

      

      

        
        
      

      

      

    

    

    

  


  
  

    STEP 4 · 수치 시각화
    
📊 숫자 → 인포그래픽 (8종)

    
본문 속 수치를 한눈에 들어오는 그림으로. 유형을 고르고 데이터를 넣으면 즉시 SVG가 그려지고, PNG·SVG로 내려받을 수 있어요.

    

      

      

    

    

    

      

    

      
      ⚠️ 의학 수치는 반드시 실제 출처를 확인하세요. 없는 숫자는 만들지 않습니다.
    

    


    

      
🖼️ AI 대표 이미지 생성 (DALL·E / gpt-image)

      
원고에 어울리는 대표 이미지·일러스트를 AI로 만들어요. 다운로드는 베놈 사이트와 동일한 WebP(품질 85)로 최적화됩니다. 프롬프트는 입력값에서 자동으로 초안이 잡혀요. ⚙️ 엔진 연결 설정(STEP3과 공유)

      
      
      

        

        

      

      
      

        
        
      

      

      

      
💡 엔진 연결 없이도 프롬프트를 복사해 미드저니·DALL·E 등 어떤 이미지 AI에든 바로 쓸 수 있어요.

    


    

      
📰 카드뉴스 텍스트 입히기

      
생성한 이미지나 직접 올린 사진에 제목·부제를 얹어 카드뉴스를 만들어요. 모두 브라우저에서 즉시 처리되고 WebP로 저장됩니다.

      

        
        
      

      
      
      
      

        

        

      

      
      
      

      

        
        
      

    


    

  


  
  

    STEP 5 · 검수 & 내보내기
    
🚦 안전등 · 품질 점수 · 브리프

    
초안을 붙여넣으면 ① 의료광고 위험 표현을 잡고 ② 글의 품질을 자동 채점하고 ③ 구어/문어 균형까지 보여줍니다.

    
    
텍스트를 입력하면 검사합니다.

    


    

      
📈 품질 점수

      

        
        

      

      
🗣️ 구어체 ↔ 문어체 균형

      

    


    

      
      
      
    

    
브리프에는 입력값·옵션·주제·타겟·페르소나·올인원 프롬프트·인포그래픽 사양이 한 파일로 담깁니다.


    

      
🟢 WordPress 자동 임시저장 (공식 API·합법)

      
내 워드프레스에 임시저장(draft)으로 자동 등록합니다. 공식 Application Password를 쓰므로 약관 위반이 아니에요. 검토 후 워드프레스에서 발행하면 됩니다. ⚙️ 워드프레스 연결

      

        

          

          

        

        
        
        

        
⚠️ API 주소(STEP3 ⚙️)도 설정돼 있어야 합니다(브라우저→내 서버→워드프레스 경유). 키는 내 서버로만 전송돼요.

      

      
      

        

        

      

      

        
        본문은 베놈 엔진 생성 결과(있으면) 또는 위 검수창 내용을 사용합니다.
      

      

    


    

      
📋 네이버·티스토리 붙여넣기용 내보내기 (자동화 대신 안전)

      
자동 등록은 약관 위반이라, 에디터에 바로 붙는 서식 HTML로 내보냅니다. 복사 → 에디터 붙여넣기 → 임시저장만 하면 끝(계정 리스크 0).

      

        
        
        
      

    


    

      
📚 여러 글 일괄 처리 (베놈 엔진 필요)

      
키워드를 한 줄에 하나씩 넣으면, 현재 카테고리·타겟·톤으로 순서대로 원고를 생성하고, 원하면 WordPress 임시저장까지 자동으로 이어서 처리해요.

      
      

        
        
        STEP3 ⚙️ API 연결이 필요합니다. 항목마다 순차 처리돼요.
      

      

    


    

  






  

    
    
❓ 원고 스튜디오 사용법

    
기획 → 집필 → 이미지/카드뉴스 → 키워드 → 검수 → 발행을 한 흐름으로. 처음이라면 우측 상단 🎲 예시부터.

    
1) 6단계 흐름

    
    
      
  • STEP 0 시작 — 카테고리·키워드·지역·글유형·톤 옵션. 브랜드색·다크모드
    • 
      
  • STEP 1 아이디어 — 주제 후보·제목 변형, 키워드 리서치(검색량)·본문 키워드 추출
    • 
      
  • STEP 2 타겟 — 직장인/환자/보호자 등 → 페르소나 자동완성
    • 
      
  • STEP 3 톤·프롬프트 — 톤 믹서 → 올인원 프롬프트 자동조립 / 베놈 엔진 생성
    • 
      
  • STEP 4 시각화 — 인포그래픽 8종·AI 이미지·카드뉴스 텍스트
    • 
      
  • STEP 5 검수·발행 — 안전등·품질점수·WordPress 발행·일괄처리
    • 
    

    
2) 서버 없이 바로 되는 것

    
✅ 프롬프트 조립·복사 · 인포그래픽 · 카드뉴스 · 키워드 추출 · 안전등 · 품질점수 · 브리프

    
3) 서버(배포+키)가 필요한 것

    
🌐 원고 자동생성 · AI 이미지 · 검색량 조회 · WordPress 발행
→ STEP 3 ⚙️ 연결 설정에 API 주소 + ADMIN_SECRET 입력(브라우저에만 저장). 엔진이 없으면 '원고 생성'이 프롬프트를 대신 복사해 줍니다.

    
4) WordPress 발행 (합법 자동화)

    
WP 관리자 → 사용자 → 프로필 → 응용프로그램 비밀번호 생성 → STEP 5 ⚙️ 연결 → 임시저장 / 예약 / 즉시 발행. (네이버·티스토리는 약관상 자동등록 대신 '붙여넣기용 복사')

    
5) 팁
